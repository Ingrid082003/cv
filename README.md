# cv

Ce projet est un test permettant de créer et visualiser un Curriculum Vitae (CV) interactif. L’objectif est de démontrer mes compétences techniques et ma capacité à mettre en place une application simple mais fonctionnelle pour la gestion et la présentation d’un CV.

Fonctionnalités
Affichage d’un CV interactif via une interface web.
Sections classiques du CV : Expérience, Formation, Compétences, Projets.
Navigation fluide entre les sections.
Possibilité de télécharger le CV au format PDF (optionnel selon implémentation).
Stack
Python – Langage principal pour la logique de l’application.
Streamlit – Pour créer une interface web simple et interactive.
Git & GitHub – Pour le versioning et le partage du projet.
HTML / CSS – Pour personnaliser l’apparence de l’interface (optionnel selon besoins).
Installation
Cloner le projet :
git clone https://github.com/ton-utilisateur/CV.git
Installer les dépendances :
pip install -r requirements.txt
Lancer l’application :
streamlit run app.py
Ouvrir le navigateur et accéder à l’URL indiquée par Streamlit.
